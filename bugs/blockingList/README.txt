In this problem, you will find 4 go files named 1 through 4. You will work through these in numerical order. These pieces of code try to implement a thread safe list with a blocking remove call. Now these 4 implementations may or may not have bugs. Your assignment is to identify what the bug is if there is a bug, or determine that there is no problem.
